### A类作业-游戏玩法（详情见开发手记）
* WASD/↑↓←→键控制人物移动
* 鼠标移动控制视角变化
* 空格键跳跃
* 鼠标左键攻击
* Esc暂停并调出游戏菜单
* 主角共5生命值，在生命值耗尽前抵达飞船上，点亮宝石即为胜利
