### 文件说明
* 3CGAME_in_3D：3D迷宫游戏。控制主角打倒怪物，触发机关，前往飞船即可获得胜利。
* 3CGAME_in_2D：2D平台跳跃游戏。控制主角跳到最高处。
* 开发手记：这次主要在做3D的迷宫游戏，2D的平台游戏是练手的，都有3C框架就一起放上来了，详见pdf。
### 3D迷宫游戏玩法
* WASD/↑↓←→键控制人物移动
* 鼠标移动控制视角变化
* 空格键跳跃
* 鼠标左键攻击
* Esc暂停并调出游戏菜单
* 主角共5生命值，在生命值耗尽前抵达飞船上，点亮宝石即为胜利
### 2D平台游戏玩法
* 左右键控制人物移动
* 空间键使女孩跳跃
* 长按空格键女孩跳得更高
* 跳到地图的最上端叭~
